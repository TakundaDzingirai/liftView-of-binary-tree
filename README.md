# liftView-of-binary-tree
You asked to print the leftview of a tree
Left view of a Binary Tree is set of nodes visible when tree is visited from left side.
Input : 
                 1
               /   \
              2     3
             / \     \
            4   5     6             
Output : 1 2 4

Input :
        1
      /   \
    2       3
      \   
        4  
          \
            5
             \
               6
Output :1 2 4 5 6
